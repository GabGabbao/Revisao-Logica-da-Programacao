O que foi requisitado:  

5. Controle de Ponto em um Escritório
Um escritório utiliza um sistema para registrar horários de entrada e saída de funcionários. As regras de controle são:

* O horário de entrada deve ser entre 8h e 9h OU o funcionário será considerado atrasado.

* O horário de saída deve ser entre 17h e 18h OU será considerado saída antecipada.

* Crie um programa que permita registrar as entradas e saídas de até 20 funcionários usando um while e que informe: 

○ Quantos chegaram atrasados.

○ Quantos saíram mais cedo.


  
14. Controle de Estoque em um Armazém
Um armazém armazena mercadorias de diferentes categorias: alimentos, produtos eletrônicos e roupas. Escreva um programa que:

* Utilize um menu (switch case) para permitir ao usuário inserir mercadorias em cada categoria ou consultar a quantidade total de mercadorias.

* O programa deve usar um while para manter o menu funcionando até que o usuário opte por sair.

* Adicione uma condição para alertar caso uma das categorias exceda 1.000 itens no estoque (com if).

* Use operadores AND para verificar se há espaço disponível e se os dados inseridos são válidos.



12. Controle de Temperatura em Máquinas Industriais
Uma máquina industrial opera em temperatura ideal entre 100°C e 200°C. Escreva um programa que:

* Permita registrar a temperatura de 10 leituras consecutivas usando um do while.

* Alerta se 3 ou mais leituras consecutivas estão fora do intervalo ideal.

* Use if e else para validar cada leitura e operadores AND para verificar as condições.



13. Gerenciador de Parcelamento Financeiro
Escreva um programa para simular o cálculo de parcelas de um empréstimo. Ele deve:

* Perguntar o valor total do empréstimo e o número de parcelas.

* Utilizar switch case para determinar as taxas de juros:

○ 3% para até 12 parcelas.

○ 5% para 13 a 24 parcelas.

○ 8% para 25 a 36 parcelas.
